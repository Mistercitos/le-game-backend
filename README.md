# 🎮 Le Game - API de Tienda de Videojuegos Digitales

## 📌 Descripción
Le Game es una API RESTful para una tienda de videojuegos digitales, desarrollada con Node.js, Express y MongoDB. Permite la gestión de usuarios, productos, carritos de compra y compras, con autenticación JWT y autorización basada en roles.
